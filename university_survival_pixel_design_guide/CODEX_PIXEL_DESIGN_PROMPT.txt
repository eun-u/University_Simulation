React + TypeScript + Vite로 만든 “한 학기만 버텨라: 대학생 생존 RPG”의 UI를 픽셀 게임 스타일로 다시 구현해줘.

중요:
기존 플랫/파스텔 앱 디자인으로 만들지 말고, 완전히 픽셀 모바일 게임 UI처럼 만들어줘.
채팅형 구조는 유지하지만, 카드/버튼/말풍선/결과창/HUD는 모두 픽셀 RPG UI처럼 보여야 해.

디자인 핵심:
- 다크 네이비 + 로열 퍼플 + 골드/민트/코랄 포인트
- 8비트/16비트 픽셀 게임 느낌
- 두꺼운 4px 검정 테두리
- blur 없는 hard shadow
- pixel font 사용
- image-rendering: pixelated
- 둥글고 부드러운 현대 앱 UI 금지
- glassmorphism / soft shadow / flat SVG 아이콘 느낌 금지

컴포넌트 이름:
- PixelAppShell
- PixelHudHeader
- PixelStatHud
- PixelMeter
- PixelWeekMap
- PixelEventScene
- PixelChatWindow
- PixelChatBubble
- PixelChoiceButton
- PixelResultPanel
- PixelEffectChip
- PixelWeekSummary
- PixelBadge
- PixelFinalResultCard

CSS:
- pixel-design-tokens.css의 변수를 사용해줘.
- 모든 주요 패널에는 pixel-panel 스타일을 적용해줘.
- 선택지 버튼은 choice-button 스타일을 적용해줘.
- 결과 변화량은 effect-chip으로 보여줘.
- 스탯바는 pixel-meter로 보여줘.

레이아웃:
- 모바일 우선
- 최대 폭 480px
- 데스크톱에서는 중앙에 게임 화면처럼 배치
- 배경은 밤 캠퍼스 느낌의 다크 네이비/퍼플
- 상단은 일반 헤더가 아니라 게임 HUD
- 중간은 채팅 로그 창
- 하단은 커맨드 선택지 버튼

에셋:
- PNG pixel sprite를 우선한다고 가정해 assetMap을 만들어줘.
- 실제 이미지가 없으면 임시로 CSS pixel placeholder 또는 emoji를 쓰되, UI 전체는 픽셀 게임 느낌이 나야 해.
- 이미지에는 pixel-art 클래스를 적용해줘.

접근성:
- 선택지는 button 태그
- focus-visible 스타일
- 색상만으로 상태 전달 금지
- 멘탈 위험, 출석 위험 같은 텍스트 표시 포함

목표:
앱을 처음 봤을 때 “채팅 앱”이 아니라 “대학생활을 픽셀 RPG로 만든 게임”처럼 보여야 한다.
