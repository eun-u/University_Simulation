# Asset Style Guide

## 공통
- 라운드 사각형, 두꺼운 라인, 밝은 파스텔 배경
- 채팅 로그 화면에 잘 붙는 간단한 실루엣
- SVG를 우선 사용하고, PNG는 보조로 사용

## 권장 UI 배치
- messageAvatar: 40~56px
- statIcon: 20~28px
- eventIcon: 36~56px
- weekBanner: 카드 상단 100% width, 120~180px height
- titleBadge: 최종 결과/획득 칭호 영역
- finalResultCardBackground: 공유 카드 배경

## 수정 방향
- 앱 색상이 정해지면 SVG의 배경색과 accent color만 일괄 교체
- 캐릭터 일러스트 확장 전까지는 이 에셋을 placeholder로 사용
